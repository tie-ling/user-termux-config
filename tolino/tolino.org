* use 15.3.2 firmware
https://download.pageplace.de/ereader/15.3.2/OS44/update.zip

mount /dev/sda /mnt
cp 15-3-2update.zip /mnt/update.zip
eject /dev/sda

reboot device to commence update

* enable adb debug the device
https://allesebook.de/anleitung/tolino-page-2-shine-3-vision-5-root-adb-und-apps-installieren-anleitung-982951/

extract custom-boot.img

execute fastboot, wait for device
fastboot flash boot.img

while connect to computer, power on by holding power button for 30 secs

fastboot reboot

* koreader config
default font: libertinus serif
screen, screen dpi: 240
e-ink settings: never refresh
disable embedded style by default, (set default by long press)
device, device status alerts, battery level: 20%, 75%

https://github.com/koreader/koreader/wiki/Android-tips-and-tricks#tolino
Create file /koreader/settings/event_map.lua and put the following content inside it:

return {
   [21] = "LPgBack",
   [22] = "LPgFwd",
}

settings, status bar, progress bar: disable, status bar items: show
all seleted items at once, auto refresh
